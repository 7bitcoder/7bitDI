Welcome to 7bitDI's documentation!
========================================

.. image:: _static/logo.svg

C++17 simple dependency injection library!

7bitDI_ is a simple C++ dependency injection library, 
designed to be as easy to use as possible, 
the main inspiration was the asp net core dependency injection system.
